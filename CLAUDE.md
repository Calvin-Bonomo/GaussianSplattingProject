## Project

A machine learning class project exploring Gaussian Splatting for real-time, high-fidelity 3D scene visualization. The core idea is rasterizing splats from a single buffer rather than path-tracing geometry, enabling real-time rendering of complex scenes.

**Planned scope:**
1. Gaussian splatting algorithm with a built-in training viewer
2. SOTA optimizations
3. Novel deep-learning optimization techniques

The repository is in early stages — no source code, build system, or dependencies have been added yet. Update this file as the project takes shape.
